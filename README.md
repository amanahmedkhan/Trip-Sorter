# Trip Sorter
node version require 6.0 or greater

## Features
- CSS
- Scss
- Stylus
- Autoprefix
- Auto Reload
- Use `require`
- Use `module.exports`
- ES6 syntax
- Remove `console.log` for production build

## Installation
`npm install`
`npm install webpack-dev-server -g`

## Run Project
`npm run server`
